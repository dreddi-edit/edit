# Dashboard (Web App)

Diese App ist die reine Web-App (Vite + React). Mobile wurde in einen separaten Ordner ausgelagert.

## Schnelle Befehle

- Web build:

```bash
npm run build
```

- Development:

```bash
npm run dev
```

- Lint + Typecheck:

```bash
npm run lint
npm run typecheck
```

## Mobile Hinweis

Alle Mobile-Befehle laufen jetzt in `apps/mobile`:

```bash
cd ../../mobile
npm run sync
npm run ios:open
npm run android:open
```
